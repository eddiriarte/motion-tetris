Welt

Wir geben unseren Spielfeld die bezeichnung Welt. Unsere Welt besteht aus einer zweidimensionale Matrix.

Daten und Darstellung strikt getrennt, das hilft uns die Übersichtlichkeit zu bewahren. Dazu verwenden wir zwei Funktionen, die einer kann eine Welt mit eine beliebige Größe erstellen und die andere übertragt den Zustand auf dem Browser.
